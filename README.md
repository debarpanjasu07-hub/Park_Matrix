# Park_Matrix
A simple, fully responsive, and scalable website for displaying real-time parking availability in a given area.

## Features

-  **Real-time Parking Availability** - View parking spots that update every 5 seconds
-  **Search Functionality** - Search parking lots by name or location
-  **Smart Filtering** - Filter by availability status (All, Available, Full, Low Availability)
- **Statistics Dashboard** - View total lots, available spots, occupancy rates
-  **Fully Responsive** - Works seamlessly on mobile, tablet, and desktop devices
-  **Modern UI** - Clean, intuitive interface with smooth animations

## Getting Started

1. Clone or download this repository
2. Open `index.html` in your web browser
